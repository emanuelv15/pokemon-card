## 💻 Getting started

```bash
# Clone this repository
$ git clone https://github.com/emanuelv15/pokemon-card.git

# Enter the folder
$ cd pokemon-card

# Install the dependencies
$ yarn

# To start the project
$ yarn start

# The app will start up in <http://localhost:3000>

```
